### Optimization Notes

Current stamp images are 1024x1024 at ~1MB each. For marketing site:
- **Hero stamps:** Export at 512x512 or 400x400, WebP format (~50-100KB)

- **Thumbnails:** Export at 128x128 if needed for grids

## 8. Asset Usage Quick Reference

### How stamps are loaded in the app

```typescript
// src/utils/clusterAssets.ts
import { getClusterAssets } from "@/utils/clusterAssets"

const assets = getClusterAssets("gamers")
// Returns:
// {
//   icon: "/images/stamp-icons/gamers.svg",
//   mainImage: "/images/STAMPS/GAMERS/MAIN.png",
//   stampImages: ["/images/STAMPS/GAMERS/NICHE/..."],
//   color: "#F2FD00"
// }
```

### Rendering stamp images

```tsx
// Standard <img> usage
<img
  src="/images/STAMPS/GAMERS/MAIN.png"
  alt="Gamers stamp"
  style={{ borderRadius: 12, width: 178, height: 178, objectFit: 'cover' }}
/>

// With Next.js Image (in app)
import Image from "next/image"
<Image
  src="/images/STAMPS/GAMERS/MAIN.png"
  alt="Gamers stamp"
  width={178}
  height={178}
  style={{ borderRadius: 12 }}
/>
```

### Rendering category icons

```tsx
import { getClusterIcon } from "@/utils/clusterAssets"

// Returns <img> element
const icon = getClusterIcon("gamers", 20)

// Or directly
<img
  src="/images/stamp-icons/gamers.svg"
  alt=""
  width={20}
  height={20}
/>
```
